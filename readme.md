# 💫 About Me:
Hey, I'm a Computer Science undergraduate driven by a strong passion for building scalable, high-performance, and impactful digital systems. I’m not just focused on learning technologies — I focus on applying them to solve real-world problems and create meaningful products.<br>My primary domain is full-stack web development, where I work across the entire stack — from crafting responsive and engaging user interfaces to designing robust backend architectures. I mainly work with JavaScript ecosystems (Node.js, Express, React, Next.js) along with databases like MongoDB, PostgreSQL, and modern cloud platforms.<br>I enjoy building real-time and system-driven applications. One of my key areas of interest is developing platforms that involve live communication and synchronization, such as video conferencing systems and interactive web apps. Working on such projects has given me hands-on exposure to concepts like WebRTC, Socket-based communication, and handling complex issues like audio feedback, latency, and real-time state management.<br>Beyond development, I’m deeply interested in system design and backend engineering. I actively explore concepts like authentication (JWT, RBAC), API architecture, database optimization, and scalable deployment strategies using platforms like AWS, Vercel, and Cloudflare. My goal is to build applications that are not only functional but also production-ready and scalable.<br>I also have a growing focus on integrating AI into applications. Whether it's building intelligent chat systems, automating processes, or enhancing user experiences through AI-driven features, I aim to combine modern development with emerging technologies to create smarter and more efficient systems.<br>What sets me apart is my approach to learning and building — I prefer understanding things from the ground up rather than relying on abstractions. I believe in writing clean, maintainable code, optimizing performance, and designing intuitive user experiences.<br>Apart from development, I also explore UI/UX design, motion graphics, and creative tools like Figma and Adobe Suite, which helps me bridge the gap between functionality and aesthetics.<br>Currently, I’m focused on:<br>* Building production-level full-stack applications<br>* Improving system design and backend architecture skills<br>* Exploring AI integrations in web applications<br>* Contributing to meaningful and innovative projects<br>I’m always open to collaboration, learning new technologies, and working on ideas that challenge me and push my limits.<br>If you're into building cool, impactful tech — let's connect and create something amazing 🚀
